# Free Deployment Options

## GitHub Pages (Recommended)
1. Push to GitHub repository
2. Go to Settings > Pages
3. Select "Deploy from a branch"
4. Choose main branch and / (root)
5. Your site will be live at: `https://[username].github.io/content-optimizer`

## Netlify (Free)
1. Drag and drop the `index.html` file to https://netlify.com/drop
2. Get instant live URL
3. Connect to Git repo for automatic updates

## Vercel (Free)
1. Sign up at https://vercel.com
2. Import GitHub repository
3. Automatic deployment on every push

## Firebase Hosting (Free)
1. Install Firebase CLI: `npm install -g firebase-tools`
2. Run: `firebase init hosting`
3. Deploy: `firebase deploy`

## Local Testing
Just double-click `index.html` to open in your browser - no server needed!

## Custom Domain
Once you have traffic, you can add a custom domain for free on any platform above.
